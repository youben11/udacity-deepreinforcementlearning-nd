# Grid World

The environment is implemented under grid_world.py and the Monte Carlo method is used in the mc.py which will learn a policy and print it to the screen, you just need to run the mc.py script.

```bash
$ python3 mc.py
[*] Learned Policy:
	[+] BOTTOM_LEFT ---> UP
	[+] TOP_LEFT ---> RIGHT
	[+] TOP_RIGHT ---> DOWN
```
